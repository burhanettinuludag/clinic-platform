# /a11y — Erişilebilirlik Denetimi

Norosera web sitesinin WCAG 2.1 AA uyumluluğunu denetle.

## Adımlar:

### 1. Otomatik Tarama
```bash
cd frontend
npx pa11y http://localhost:3000 --standard WCAG2AA --reporter cli
npx pa11y http://localhost:3000/hizmetler --standard WCAG2AA --reporter cli
npx pa11y http://localhost:3000/blog --standard WCAG2AA --reporter cli
npx pa11y http://localhost:3000/iletisim --standard WCAG2AA --reporter cli
```

### 2. Manuel Kontrol (sub-agent)
Spawn sub-agent: Frontend dosyalarını tara
- `<img>` ve `<Image>` elementlerinde `alt` text var mı
- `<button>` elementlerinde `aria-label` veya görünür text var mı
- `<input>` elementlerinin `<label>` eşleştirmesi var mı
- Renk kontrastı yeterli mi (Tailwind class'larından kontrol)
- `tabIndex` düzeni mantıklı mı
- Skip-to-content link var mı

### 3. Form Erişilebilirliği
```bash
# Formları bul ve kontrol et
grep -rn "<input\|<select\|<textarea" frontend/src/ --include="*.tsx" | \
    grep -v "aria-\|htmlFor\|label"
```

### 4. Sonuç
WCAG 2.1 AA uyumluluk oranını hesapla ve düzeltme öncelikleri belirle.
